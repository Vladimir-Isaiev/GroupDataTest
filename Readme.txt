Настройка подключения к SQLServer:
В файле GroupDataTest.txt запишите имя сервера, имя экземпляра, логин и пароль в соответствующие строки. Вводимые имена должны быть последними в строке, пробелы после них не допускаются.
Например:

Server = DESKTOP-SRETNG2
Instance = SQLEXPRESS
DataBase = shipmentdb
UserId = ttest
Password = 90909090
